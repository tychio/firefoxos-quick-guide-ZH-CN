# Firefox OS快速开发指南

该项目为英文书籍"Quick Guide for Firefox OS Development"的中文Markdown翻译资源。

## 原稿页面

书籍英文原稿可以从Leanpub获取，这是其[发布页面](https://leanpub.com/quickguidefirefoxosdevelopment/)。
